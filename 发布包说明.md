## 💼 使用场景示例

向用户展示实际应用：

### 1. API 集成
```sql
-- 调用第三方 API
SELECT http_get('https://api.github.com/users/octocat');
```

### 2. Webhook 通知
```sql
-- 发送 Slack 通知
SELECT http_post(
    'https://hooks.slack.com/...',
    '{"text":"订单已创建"}'
);
```

### 3. 数据同步
```sql
-- 从外部 API 导入数据
INSERT INTO products (name, price)
SELECT 
    (data->>'name')::text,
    (data->>'price')::numeric
FROM (
    SELECT http_get('https://api.example.com/products')::json AS data
) t;
```
